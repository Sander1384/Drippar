# Driparr Public Beta Promotion Posts

Use these short posts when asking for early feedback from Radarr and self-hosted communities.

## English

```text
I built Driparr, a small self-hosted app for Radarr users who want to import IMDb CSV lists gradually instead of dumping a huge list into Radarr all at once.

It lets you import an IMDb CSV, review the queue, choose a drip mode, and let it add movies in controlled batches. It also has duplicate protection, a live status feed, Docker/Portainer support, and a mock Radarr test stack so you can try it safely first.

I am looking for early testers from the Radarr/self-hosted community.

GitHub:
https://github.com/Sander1384/Drippar
```

## Core Message

```text
Driparr is a controlled IMDb CSV drip-feeder for Radarr.

It helps you process large IMDb exports gradually, visibly, and safely instead of flooding Radarr with a huge list at once.
```
